# How to Move Code to Different Branch:
| Action                   | Command                       |
| ------------------------ | ----------------------------- |
| Switch to source branch  | `git checkout main`           |
| Pull latest updates      | `git pull origin main`        |
| Switch to target branch  | `git checkout Development`    |
| Merge source into target | `git merge main`              |
| Push merged code         | `git push origin Development` |
